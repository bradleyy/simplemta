==========
SimpleSMTP
==========

Requirements
------------

* Eventlet
* Greenlet

Usage
-----

  ./simplesmtp.py start		Starts the server

  ./simplesmtp.py stop		Stops the server

  ./simplesmtp.py restart	Restarts the server
